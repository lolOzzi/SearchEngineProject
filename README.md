# SearchEngineProject

When starting the program it is important to run the add file command. After this you can query the program with a string or a do a regex query to get the Wikipedia Articles either the string or the regex query appears in. 
The program expects the datafile to be in the westburylab.wikicorp format.

This is the bachelor project done by Christian Nkya (s236137) , Oliver Svanholm Kliim (s235278) and Oscar Stilling (s235281). 
It is made with C++ 20. It can be compiled using the CMakeLists file.
